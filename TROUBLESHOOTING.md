# 🔍 错误诊断指南

## 请告诉我你看到的情况

### 在左侧输入 Markdown 后，右侧出现了什么问题？

**选项 A：完全没有显示**
- 右侧是空白
- 或者只显示占位符"粘贴内容预览"

**选项 B：显示了原始内容**
- 右侧显示的是 Markdown 源码（如 `# 标题`）
- 没有被解析成 HTML

**选项 C：显示的样式错误**
- 内容显示了但样式不对
- 主题没有应用
- 布局混乱

**选项 D：其他错误**
- 请描述具体现象

---

## 快速测试步骤

### 1. 打开浏览器开发者工具
- Mac: `Cmd + Option + I`
- Windows: `F12` 或 `Ctrl + Shift + I`

### 2. 切换到 Console 标签
- 查看是否有红色错误信息
- 截图或复制错误信息

### 3. 切换到 Network 标签
- 刷新页面
- 查看 marked.js、themes.js、app.js 是否成功加载（状态码 200）

---

## 临时解决方案

如果是页面刚打开就有问题，尝试：

1. **清空编辑器**
   - 点击"🔄 重置"按钮
   - 或按 `Cmd+K`

2. **手动输入简单内容测试**
   ```
   # 测试标题

   这是测试内容
   ```

3. **检查浏览器控制台**
   - 看是否有 JavaScript 错误
